# [게임 구성 요소]

## [스토리]

주인공은 개 왕국의 공주

마법 능력을 가지고 태어남.

그러나 점점 마력이 약해져서 평범한 사람과 같아짐

<br>

어느날 보물창고에서 이상하게 끌리는 보석을 발견하고 그것을 만졌는데 빛이 쏟아져나옴.

그리고 마력이 돌아온 것을 느낌.

그날 밤 주인공은 꿈을 꾸게 되는데 1년 후 이웃나라와 전쟁이 발발했는데 왕국이 패배하는 꿈이었음.

설상가상으로 사랑하는 언니가 포로로 잡혀감.

<br>

꿈에서 깬 주인공은 이것이 예지몽이라는 것을 알아차림.

(마법 능력을 되찾아 예지몽을 꾼 것임)

<br>

이 사실을 모두에게 알리려고 시도해봤지만 관련된 말을 꺼내려고 하면 목소리가 나오지 않아 실패함.

도움을 받을 수 없다는 것을 받아들인 주인공은 자신의 힘으로 왕국과 포로가 된 언니를 전쟁으로부터 구하기로 결심함.
<br>

## [메커니즘]

주인공은 매주 기술을 올려주는 수업을 들어야 함.

주인공이 배운 기술들에 따라, 게임을 진행하는 동안 나오는 선택지가 변함.

어떤 기술을 배우냐에 따라 볼 수 없는 이벤트도 있음.

<br>

수업을 하면 스트레스가 일정치 쌓임

스트레스 수치가 일정수준 이상 올라가면 캐릭터가 수업을 제대로 수행할 수 없음

스트레스 관리에 주의

<br>

npc 호감도는 주인공이 배울 기술이나 이벤트에 영향을 줌.

npc에 따라 기술을 배울 때 주는 보너스나 페널티가 바뀜.

친해진 npc에 따라 열리는 선택지가 있음.

<br>

기술목록

( 개 능력 / 신체 능력 / 마법 능력 / 지식 / 신성력 )

<br>

일주일에 2개의 기술을 배울 수 있음. 주말을 제외한 하루마다 배우려 하는 기술을 10포인트씩 획득 (제한 1000)

<br>

주말에는 마을에 방문해서 npc의 호감도를 올리거나 능력치를 올릴 수 있음

특정 주차에는 숨겨진 이벤트 스토리를 볼 수도 있음 / 토,일 오전오후 4번

<br>

이야기를 진행하면서, 시스템 안내메세지창이 왼쪽 상단에 나타날 때가 있음.

이 안내메세지들은 장면 이면에 무슨 일이 일어나는지 알려줌.(능력치 or 호감도 수치의 변화)

<br>

또 다른 메세지는 플레이어가 이전에 배운 기술에 의해 이야기가 바뀔 때 보이게 됨.

만약 실패했다는 메세지가 나오면, 해당 기술이 요구 수치에 못 미쳤기 때문에 숨겨진 선택지를 지나치게 된다는 뜻.

<br>

이러한 메세지들은 숨겨진 다른 선택지들을 알아내고 다음 회차 플레이에 도움을 줄 수 있음.

<br>

예를 들어 산책을 하고 있었는데 도둑을 발견함 -> 경비병에게 신고한다, 도망친다, (쫓아간다) <- 숨겨진 선택지

[민첩성 < 30이면 안내창에 민첩성 -> 실패 라는 메세지가 뜸] 

<br>

## [게임의 규칙]

### 1) 핵심 규칙

* 플레이어는 스케쥴 설정창에서 자신이 배울 능력치를 선택할 수 있다.
* 능력치를 올려 대관식까지 여왕의 자격을 갖춰야한다.


### 2) 보조 규칙

* 스트레스가 100이 되면 게임오버
* 게임상으로 1년이 지나면 게임이 끝난다. 
* 스토리 분기점에서 선택지 선택에 따라 엔딩이 달라진다.
* NPC는 주말에 방문하는 마을에서 만날 수 있는데 주차마다 장소가 달라진다.

## [게임에서 사용될 공식]

### (1) 능력치 >= 필요수치  
* 능력치 선택지 등장
<br>  

### (2) 스트레스 > 30
* 능력치 효율 / 2

### (3) 스트레스 > 70
* 능력치 효율 / 5


## [요구사항]

* 시작 화면, 엔딩모음 화면, 메인 화면, 능력치 화면, 스케쥴 화면, 인벤토리 화면, 저장 화면, 스토리 화면, 마을 화면, 엔딩 화면 총 10개의 화면이 있다.
* 시작 화면에는 게임시작, 엔딩모음이라는 버튼 2개와 로고가 있다.
* 엔딩모음 클릭 시 엔딩모음 화면으로 이동한다.
* 엔딩모음 화면에는 엔딩이 최대 4개까지 표시되고 화살표 버튼으로 페이지를 이동할 수 있다.
* 엔딩모음 화면에는 시작 화면으로 가는 버튼이 있다.
* 게임시작 클릭 시 게임화면으로 이동한다.
* 게임의 종료 조건은 플레이어가 사망하거나 12주차(미정)에 도달했을 때 종료한다.
* 메인 화면에는 능력치, 스케쥴, 인벤토리, 저장이라는 버튼이 있다.
* 메인 화면에는 현재 상태(주차와 스트레스 수치)가 오른쪽 위에 표시된다.
* 메인 화면의 가운데에 캐릭터의 현재 모습이 표시된다.
* 메인 화면에서 능력치 버튼을 클릭 시 능력치 화면으로 이동한다.
* 능력치 화면에는 현재 능력치 수치와 호감도 수치 창이 표시되고 뒤로가기 버튼이 있다. 뒤로가기 버튼 클릭 시 메인 화면으로 이동한다.
* 메인 화면에서 스케쥴 버튼을 클릭 시 스케쥴 화면으로 이동한다.
* 스케쥴 화면에는 현재 수행할 수 있는 스케쥴 목록과 달력이 표시된다.
* 스케쥴 목록에서 스케쥴 클릭 시 첫주부터 순서대로 달력에 배치되고 달력에서 스케쥴 우클릭 시 해당 스케쥴이 삭제된다.
* 스케쥴 화면에는 확정 버튼과 뒤로가기 버튼이 있다. 확정 버튼 클릭 시 스케쥴 실행 팝업창이 표시되고 현재 실행중인 스케쥴의 애니메이션이 실행된다. 뒤로가기 버튼 클릭 시 메인 화면으로 이동한다.
* 스케쥴 실행 후 스토리가 있는 주차라면 스토리 화면으로 이동한다.
* 메인 화면에서 인벤토리 버튼을 클릭 시 인벤토리 화면으로 이동한다.
* 인벤토리 화면에는 하단에 아이템 목록이 표시된다. 일반 아이템과 의상탭이 있다. 일반 아이템 탭에는 스트레스 아이템, 능력치 아이템이 표시되고 의상 탭에는 의상만 표시된다. 뒤로가기 버튼이 있다. 뒤로가기 버튼 클릭 시 메인 화면으로 이동한다.
* 메인 화면에서 저장 버튼을 클릭 시 저장 화면으로 이동한다.
* 저장 화면에는 저장슬롯이 6개 표시된다. 클릭 시 해당 위치에 저장할 것인지 묻는 팝업창을 띄우고 예를 누르면 저장, 아니오를 누르면 저장하지 않고 팝업창이 닫힌다. 뒤로가기 버튼이 있다. 뒤로가기 버튼 클릭 시 메인 화면으로 이동한다.
* 스토리 화면에는 하단에 대화창이 표시된다. 대화창 오른쪽 위에 안내메세지 창이 표시된다. 대화창 양쪽 위에 캐릭터 스탠딩 일러스트가 최대 2개까지 표시된다. 화면 중간에 선택지가 최대 4개까지 표시된다.
* 대화창에는 현재 말을 하고 있는 캐릭터의 이름이 왼쪽 상단에 표시된다. 대화창을 클릭 시 대화가 넘어간다.
* 안내메세지 창은 안내메세지가 있을 때만 표시된다. 안내메세지 창은 5초가 지나면 사라진다.
* 능력치 선택지는 능력치>요구치 일 때 표시된다. 능력치<요구치일 시 안내메세지 창에서 실패 알림이 표시된다.(호감도도 동일)
* 선택지 선택 시 다시 대화가 진행된다.
* 잘못된 선택지를 선택 시 캐릭터가 사망하고 엔딩 화면으로 이동한다.
* 엔딩 화면에는 이번 회차에 본 엔딩 관련 일러스트와 설명이 표시된다. 메인으로 버튼이 있다. 버튼 클릭 시 시작 화면으로 이동한다.
* 스토리 진행 중간에 미니게임이 등장한다.(간단한 클릭게임)
* 마을 화면에는 여섯개의 아이콘(성, 호수, 번화가, 뒷골목, 상점, 성당)이 있다. <br>상점 아이콘 클릭 시 아이템을 구매 화면으로 이동한다.<br>성당 아이콘 클릭 시 성당에 모금을 하는 화면으로 이동한다.<br>성을 제외한 5개의 아이콘 모두 클릭 시 랜덤하게 NPC와 대화 화면으로 이동한다.<br>성 아이콘 클릭 시 해당 주차의 외출을 종료할 수 있다.
* 외출은 마지막 주 주말에 4회 할 수 있다.

## [용어정리]

### index(시작화면) -> 프로그램 시작 시 첫 화면. 게임시작 버튼, 엔딩모음 버튼이 있다.

* 화면 요소
    - 게임시작 버튼: 클릭 시 게임 화면으로 이동한다.
    - 엔딩모음 버튼: 클릭 시 엔딩모음 화면으로 이동한다.

### ending list(엔딩모음화면) -> 수집한 엔딩을 볼 수 있는 화면이다. 페이지 이동 버튼과 뒤로가기 버튼, 엔딩 목록이 있다.

* 화면 요소
    - 페이지 이동 버튼: 클릭 시 엔딩모음 페이지가 전환된다.
    - 뒤로가기 버튼: 클릭 시 시작 화면으로 이동한다.
    - 엔딩 목록: 수집한 엔딩을 나열한 목록이다. 일러스트, 엔딩 이름이 표시된다.
* 스크립트 요소
    - EndingList: 저장된 엔딩 목록 데이터이다.

### main(메인화면) -> 능력치, 스케쥴, 인벤토리, 저장 화면으로 이동할 수 있는 화면이다. 능력치, 스케쥴, 인벤토리, 저장 버튼과 상태 UI, 플레이어 캐릭터가 있다.

* 화면 요소 
    - 능력치 버튼: 클릭 시 능력치 화면으로 이동한다.
    - 스케쥴 버튼: 클릭 시 스케쥴 화면으로 이동한다.
    - 인벤토리 버튼: 클릭 시 인벤토리 화면으로 이동한다.
    - 저장 버튼: 클릭 시 저장 화면으로 이동한다.
    - 상태 UI: 화면 우측 상단에 현재 주차와 스트레스 수치, 골드가 표시된다.
    - 플레이어 캐릭터: 화면 가운데에 플레이어 캐릭터의 현재 모습이 표시된다.
* 스크립트 요소
    - stress: 현재 스트레스 수치이다.
    - week: 현재 진행 주차이다.
    - gold: 현재 보유한 골드이다.

### status(능력치화면) -> 현재 능력치 수치를 보여주는 화면이다. 능력치, 호감도 수치 창과 뒤로가기 버튼이 있다.

* 화면 요소
    - 능력치, 호감도 수치 창: 현재 능력치 수치를 종류별로 나열한 목록이다. 0~1000/0~100 까지 게이지로 표시된다.
    - 뒤로가기 버튼: 클릭 시 메인 화면으로 이동한다.
* 스크립트 요소
    - npcFav: 현재 호감도 수치이다.
    - maxFav: 최대 호감도 수치이다.
    - statSt: 현재 능력치 수치이다.
    - maxSt: 최대 능력치 수치이다.
    - plusSt: 현재 능력치 효율 수치이다. 스트레스에 따라 변한다.

### schedule(스케쥴화면) -> 스케쥴을 설정하는 화면이다. 현재 수행할 수 있는 스케쥴 목록, 달력, 뒤로가기 버튼이 표시된다.

* 화면 요소
    - 스케쥴 목록: 현재 수행할 수 있는 스케쥴 목록이 표시된다.
    - 달력: 설정한 스케쥴이 달력에 표시된다.
    - 뒤로가기 버튼: 클릭 시 메인 화면으로 이동한다.
* 스크립트 요소
    - SchList: 설정한 스케쥴 목록 데이터이다.
    - exSch: 선택한 스케쥴의 인덱스이다.

### inventory(인벤토리화면) -> 현재 보유한 아이템을 보여주는 화면이다. 일반아이템 탭과 의상탭, 현재 플레이어 캐릭터, 현재 탭의 아이템과 뒤로가기 버튼이 표시된다.

* 화면 요소
    - 탭: 일반아이템, 의상탭이 화면 하단에 표시된다.
    - 아이템: 현재 탭의 아이템이 최대 3개/5개 표시된다.
    - 플레이어 캐릭터: 화면 가운데에 플레이어 캐릭터의 현재 모습이 표시된다.
    - 뒤로가기 버튼: 클릭 시 메인 화면으로 이동한다.
* 스크립트 요소
    - itemCount: 특정 아이템의 현재 보유 수 이다.
    - dressN: N번의 의상 아이템의 인덱스이다.
    - stressItemN: N번의 스트레스 아이템의 인덱스이다.
    - statItemN: N번의 능력치 아이템의 인덱스이다.

### save(저장화면) -> 게임의 현재 진행 상황을 저장하는 화면이다. 저장슬롯과 뒤로가기 버튼이 표시된다.

* 화면 요소
    - 저장슬롯: 게임을 저장할 수 있는 슬롯이다. 최대 6개까지 저장할 수 있다.
    - 뒤로가기 버튼: 클릭 시 메인 화면으로 이동한다.
* 스크립트 요소
    - saveN: N번의 저장 데이터이다.

### story(스토리화면) -> 메인 스토리 진행 화면이다. 대화창, 안내메세지 창, 캐릭터, 선택지가 표시된다.

* 화면 요소
    - 대화창: 화면 하단에 표시된다. 대화 스크립트가 표시되는 창이다. 현재 발언중인 캐릭터의 이름이 왼쪽 상단에 표시된다.
    - 안내메세지 창: 대화창 우측 상단에 표시된다. 안내메세지가 없을 땐 숨겨져 있다. 표시되고 5초가 지나면 숨겨진다.
    - 캐릭터: 대화중인 캐릭터이다. 최대 2명까지 표시된다.
    - 선택지: 화면 가운데에 표시된다. 최대 4개까지 표시된다.
* 스크립트 요소
    - CharacterName: 현재 대화중인 캐릭터 이름이다.
    - demandSt: 능력치 선택지 오픈에 필요한 능력치 최소 요구치이다.
    - demandFav: 호감도 선택지 오픈에 필요한 호감도 최소 요구치이다.

### town(마을 화면) -> 네번째 주 주말 외출 시 방문할 수 있는 마을 화면이다. 방문할 수 있는 장소의 아이콘이 있다.

* 화면 요소
    - 상점: 아이템을 구매할 수 있는 장소이다.
    - 성당: 기부를 할 수 있는 장소이다. 100000골드를 기부하면 특별한 아이템을 획득할 수 있다. (모든 능력치 + 300)
    - 성: 외출을 마치고 집으로 돌아가는 아이콘이다.
    - 상점, 성당, 호수, 번화가, 뒷골목: NPC를 만나 호감도를 올릴 수 있는 장소이다.
* 스크립트 요소
    - price: 아이템의 가격이다.
    - donation: 현재까지 기부한 금액이다.
    - visitCount: 외출 횟수이다. 아이콘을 클릭할 때마다 -1
