# 6주차 개발 목표

작업명: 인벤토리 기능 구현

해당 요구사항: 
*  아이템 클릭 시 사용여부를 묻는 팝업창이 표시된다. 사용한다를 선택하면 아이템 개수가 줄어들고 효과가 적용된다.
*  아이템 ui에 아이템 이미지와 설명이 표시된다.

설명: 

아이템 클릭 시 팝업창이 표시된다.  
<br>
사용한다를 선택할 시 인벤토리의 아이템이 사라지고 해당 아이템의 효과가 적용된다.  
<br>
아이템 ui 디자인  
<br>
시작 스토리 제작
