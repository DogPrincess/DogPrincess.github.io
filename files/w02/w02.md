# 2주차 개발 목표

작업명: 메인 화면 구성

해당 요구사항: 
*  메인 화면에는 능력치, 스케쥴, 인벤토리, 저장이라는 버튼이 있다.
*  메인 화면의 가운데에 캐릭터의 현재 모습이 표시된다.
*  주인공 캐릭터 디자인
*  UI 규격 설정

설명:   

메인 화면에 버튼 UI를 배치한다.  
<br>
배치할 UI 규격을 설정한다.  
<br>
메인 화면에 배치될 캐릭터의 기본 일러스트를 세부 제작한다.
